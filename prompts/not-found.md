# `/not-found` Prompt

```text
Responsive 404 / not found page for BoilerSub, matching the BoilerSub landing page and product design language.

Goal:
Create a branded not-found page that still feels premium, campus-native, and intentional rather than like a default framework fallback.

Core Function:
Gracefully handle invalid routes or missing pages and guide the user back into the core marketplace.

Required Elements:
- BoilerSub branding
- Strong editorial headline like “Page not found” with BoilerSub personality
- Short supporting message that feels human and campus-aware
- Primary CTA back to browse listings
- Secondary CTA back to home if desired
- Optional subtle visual motif or branded illustration placeholder

Content Rules:
- Keep tone polished, lightly human, and not overly playful
- Make recovery obvious and fast
- Reinforce the BoilerSub identity even on error pages

Visual Style:
- Match BoilerSub’s premium editorial visual system
- Atmospheric, modern, and calm
- Avoid plain default error-page layouts
- Use tonal depth, spacing, and strong type to make the page feel intentional

Branding and Colors:
- Off-white / bone background
- Purdue-inspired muted gold accents
- Electric blue for recovery actions
- Kinetic coral only for subtle attention accents if needed

Typography:
- Plus Jakarta Sans for the main message
- Manrope for supporting text and CTA labels

Layout Guidance:
- Desktop and mobile should both feel centered, elegant, and easy to recover from
- Keep the content focused and visually memorable

Accessibility:
- Clear action buttons
- Strong contrast
- Readable messaging and touch-friendly controls

Platform:
Responsive web for both desktop and mobile
```
